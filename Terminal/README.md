Available commands:
`help`, `clear`, `sysinfo`, `scan`, `scp`, `connect`, `reboot`, `my ID`.

`help` — display command list or full manual
&gt; Shows all available terminal commands.

`my ID` — display the account ID
&gt; Shows the ID of this account

`clear` — clear current terminal screen
&gt; Erases text buffer. Logs remain accessible.

`sysinfo` — display system information
&gt; Returns basic CORIE info, access level, time, and user id.

`scan [directory]`
&gt; Scans remote directories for accessible files and subfolders.

`scp [path]`
&gt; Copies file or directory to local workspace (/Desktop).

`connect [node]`
&gt; Attempts to establish a session with the specified node. Connection availability may vary depending on node status, access restrictions, or remote blocking rules.

`reboot` — restart CORIE interface
&gt; restart CORIE interface
